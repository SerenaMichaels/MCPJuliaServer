MCP File Server

Model Context Protocol Server Documentation

Server Information

  • Server Type: file
  • WSL IP: 172.27.85.131
  • Port: 8081
  • Base URL: http://172.27.85.131:8081

Available Tools (5)

list_directory

List contents of a directory with file details

{"properties":{"path":{"type":"string","description":"Directory path to list (relative to base directory)"}},"required":["path"],"type":"object"}

read_file

Read the contents of a text file

{"properties":{"path":{"type":"string","description":"File path to read (relative to base directory)"}},"required":["path"],"type":"object"}

write_file

Write content to a file

{"properties":{"content":{"type":"string","description":"Content to write to the file"},"path":{"type":"string","description":"File path to write (relative to base directory)"}},"required":["path","content"],"type":"object"}

create_directory

Create a new directory

{"properties":{"path":{"type":"string","description":"Directory path to create (relative to base directory)"}},"required":["path"],"type":"object"}

delete_file

Delete a file or directory

{"properties":{"path":{"type":"string","description":"File or directory path to delete (relative to base directory)"}},"required":["path"],"type":"object"}

Claude Desktop Configuration

{ "mcpServers": { "mcp-file-http": { "command": "powershell", "args": [ "-Command", "$response = Invoke-RestMethod -Uri 'http://172.27.85.131:8081/mcp/tools/call' -Method POST -ContentType 'application/json' -Body (ConvertTo-Json @{name=$args[0]; arguments=(ConvertFrom-Json $args[1])}); Write-Output ($response.result.content[0].text)" ], "env": {}, "description": "MCP File Server via HTTP" } } }

API Endpoints

  • GET /mcp/health - Health check
  • GET /mcp/info - Server information
  • GET /mcp/docs - This documentation page
  • POST /mcp/tools/list - List available tools
  • POST /mcp/tools/call - Execute a tool
